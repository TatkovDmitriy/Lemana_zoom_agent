# ADR-001: Initial Project Structure

**Date:** 2026-05-11  
**Status:** Accepted

## Context
Необходимо заложить структуру проекта для Zoom-агента до начала активной разработки.

## Decision
Разделение на модули по ответственности:
- `zoom/` — интеграция с Zoom API (получение записей)
- `transcription/` — транскрибация аудио
- `agent/` — оркестрация, LLM-логика (Claude)
- `delivery/` — доставка результатов

## Rationale
Такое разделение позволяет независимо менять провайдеров (транскрибация, LLM, доставка) без переписывания всего кода.
